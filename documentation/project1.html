Network Setup for Small Business

Overview

This project involved setting up a secure and efficient network infrastructure for a small business. The goal was to ensure reliable internet access, secure communications, and remote access capabilities for employees.

Objectives

  • Implement a scalable network solution.
  • Ensure data security through VPN and firewall configurations.
  • Provide remote access to key resources for offsite employees.

Technologies Used

  • Cisco Routers and Switches
  • OpenVPN
  • Windows Server for DHCP and DNS
  • Firewall: pfSense

Steps Taken

  1. Analyzed client’s needs and existing infrastructure.
  2. Designed a network topology that supports scalability and security.
  3. Installed and configured routers and switches.
  4. Set up a VPN for remote access.
  5. Configured the firewall to protect the network perimeter.
  6. Tested the network for performance and security vulnerabilities.

Outcome

The project was completed on time and within budget. The small business now enjoys a robust and secure network with the flexibility to support remote work.
